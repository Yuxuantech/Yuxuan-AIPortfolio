# Yuxuan-AI-Portfolio

An AI-powered professional portfolio and resume platform. This project features a polished UI, multi-language support, and a unique interactive voice introduction module.

## 🚀 Features

- **Dynamic Interactive Resume**: Clean, modern interface with smooth animations (via `framer-motion`).
- **Interactive Voice Module**: 
  - Real-voice introduction playback.
  - **Atmospheric BGM Integration**: Background music features an intelligent "Fade-in" effect that triggers 3 seconds before the voice recording ends.
  - Continuous music playback until the BGM track finishes naturally.
- **Responsive Design**: Optimized for both desktop and mobile devices.
- **Multilingual Support**: Switch seamlessly between Chinese and English.

## 🛠️ Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

## 📦 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Yuxuantech/Yuxuan-AI-Portfolio.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
